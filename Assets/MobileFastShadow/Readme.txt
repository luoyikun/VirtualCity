Mobile Fast Shadow v1.03

----------------------------------------------------------------------------------------------
1. Drag "MobileFastShadow.Prefab"  into the scene.
2. Select that you just dragged in in the scene.
3. Setting up the MobileFastShadow component.
4. "Layer Caster" represents objects with shadows, usually roles, and so on.
5. "Layer Ignore Receive" indicates that an object that ignores the received shadow is generally an object other than the ground.
6. You can create your own layer freely through Layer.
7. Remember to set the correct layer of the corresponding object in the scene.

----------------------------------------------------------------------------------------------
Matters needing attention:

-If Follow Cam is null,The shadow projector will automatically follow the Tag's "MainCamera" camera, so please set up your camera correctly.
